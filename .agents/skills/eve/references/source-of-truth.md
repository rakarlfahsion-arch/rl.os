# eve Source Of Truth

Use the documentation bundled with the installed `eve` package before editing an
eve project:

```bash
node_modules/eve/docs/README.md
```

If the package is not installed, install or scaffold it first so the docs match
the version in use:

```bash
npm install eve
npx eve init <agent-name>
```

Treat this repo-local skill as a router only. Do not copy framework guidance
from memory or older docs when the bundled docs are available.
